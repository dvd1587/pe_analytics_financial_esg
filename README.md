# Private Equity Financial & ESG Analytics

### [Tableau Demonstration](https://public.tableau.com/app/profile/dprateek/viz/PEAnalytics/Overview)

<h2>Description</h2>
An interactive executive dashboard in Tableau tracking fund KPIs and portfolio performance analytics. It features 2 dashboards, 15 visualizations and charts, and clickable portfolio company level analytics and filters. This dashbaord can be used for both the fund-level financial performance analytics along with the portfolio level deep-dive metrics and performance visualization along with tracking ESG KPIs too. The dashboard provides two ways by which the user can go to the portfolio-level analytics: 1. Clicking on the Portfolio Level Analytics navigation button, and 2. Clicking on the company name in the Overview page.
<br />

<h2>Skills and Tools Used</h2>

- <b>Tableau, Data Visualization, Financial Analysis</b>
